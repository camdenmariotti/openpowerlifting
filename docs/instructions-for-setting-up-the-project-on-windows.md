# Instructions for Setting Up the Project on Windows

Below are step-by-step instructions for setting up the project.

## Windows 10
1. Enable Linux Subsystem for Windows
   •https://docs.microsoft.com/en-us/windows/wsl/install-win10
      •Choose Ubuntu
2. Install automake
   •In Ubuntu type: `sudo apt-get update`
   •Then: `sudo apt-get install automake`
3. Install Git
   •In Ubuntu type: `sudo apt-get install git`
4. Clone Repo
   •In Ubuntu type: `git clone https://github.com/sstangl/openpowerlifting.git`
5. Install sqlite3
   •In Ubuntu type: `sudo apt-get install sqlite3`
6. Install npm
   •In Ubuntu type: `sudo apt-get install npm`
7. Install node
   •In Ubuntu type: `sudo apt-get install node.js`
8. Node stuff
   •In Ubuntu type: `sudo ln -s /usr/bin/nodejs /usr/bin/node`
9. Change directory
   •In Ubuntu type: `cd openpowerlifting`
10. Copy directory to PC
